## 📖 Project Description

ForexMaster Server is a RESTful API built to power the ForexMaster platform. It handles authentication, analysis management, database operations, and secure communication between the frontend application and MongoDB.

## ⚙️ Installation Guide

### Clone the repository

```bash
git clone https://github.com/apurbochaki-lab/ForexMaster-server.git
```

### Move into project directory

```bash
cd ForexMaster-server
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
PORT=
MONGODB_URI=
NEXT_PUBLIC_CLIENT_URL=
```

### Run the server

```bash
npm run dev
```
